## 🚀 Featured Projects

### 📊 psiwatch — Zero-Dependency Python ML Drift Detection

**Production ML monitoring, the way it should be.**

> Detects data drift between training and production datasets using PSI (Population Stability Index), Chi-Square analysis, Mean Shift detection, and Standard Deviation analysis. Pure Python with **zero external dependencies**.

| Attribute | Details |
|:--|:--|
| **PyPI** | [psiwatch](https://pypi.org/project/psiwatch/) — `pip install psiwatch` |
| **Size** | ~15KB total package size |
| **Stack** | Pure Python · Standard library only · No numpy, scipy, or pandas required |
| **Features** | CLI tool · HTML/JSON/TXT reports · `--fail-on-drift` CI flag · Webhook alerts · Baseline locking · Directory watcher · Trend analysis · Multi-dataset comparison |
| **Platforms** | Windows · macOS · Linux · Google Colab · Termux/Android |
| **Repository** | [github.com/tharunstryker/psiwatch](https://github.com/tharunstryker/psiwatch) |
| **Use Cases** | Production monitoring · CI/CD integration · Data quality gates · Model retraining triggers |

**Quick Start:**
```bash
pip install psiwatch

# Compare datasets for drift
psiwatch compare train.csv production.csv

# Generate HTML report
psiwatch compare train.csv production.csv --report html

# Fail pipeline on drift detection
psiwatch compare train.csv production.csv --fail-on-drift
```

- 20+ customizable themes with animations
